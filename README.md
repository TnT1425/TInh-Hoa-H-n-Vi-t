# 🌾 Dự án: Tinh Hoa Hồn Việt (Đặc Sản 3 Miền)

## 🚀 Hướng dẫn cài đặt và chạy dự án (Dành cho thành viên nhóm)

**Bước 1: Cài đặt thư viện**
Mở 2 Terminal riêng biệt cho Frontend và Backend, chạy lệnh sau ở cả 2 bên:
\`\`\`bash
npm install
\`\`\`

**Bước 2: Cấu hình Backend**
1. Vào thư mục \`backend-tinhhoahonviet\`.
2. Đổi tên file \`.env.example\` thành \`.env\`.
3. Điền link MongoDB và mật khẩu Email của nhóm vào file \`.env\`.

**Bước 3: Khởi động hệ thống**
- Ở Terminal Backend gõ: \`npm run dev\`
- Ở Terminal Frontend gõ: \`npm run dev\` (hoặc \`npm run start\`)

Hệ thống sẽ chạy tại: `http://localhost:5173`
